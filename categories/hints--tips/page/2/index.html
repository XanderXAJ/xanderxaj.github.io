Quickly Open an Elevated Command Prompt in Vista

 Posted on July 26, 2009

Just a quick little thing I noticed, although everyone else probably knows it by now. To launch an elevated command prompt (i.e. a prompt with admin privileges, required to do various things if UAC is enabled on Vista), simply open the Start Menu (Windows key), type “cmd” without quotes, and press Control+Shift+Return. This will bring up a UAC prompt asking you to elevate the process. Select Continue (Alt+C). Done! This does work for some other applications. [Read More]

Reset iTunes Dimensions to Fit Resolution

 Posted on June 25, 2009

Here’s a handy little feature that I find useful. If you ever play around with the size or location of the iTunes window and want to make it fit the screen better, simply follow the instructions: If you’re on Windows: hold Shift and either double-click the title bar or single-click the maximise button (The title bar is the small bar at the top that says “iTunes” in the middle.) If you’re on Mac OS: hold Option and click the Zoom (+) button This’ll reset the size and position to its default for your screen resolution. [Read More]
